# springmvc
<springmvc + mybatis + mysql>

\n\n.
项目是idea/maven 形式创建.
以springmvc + mybatis +mysql 的一个访问数据的简单程序demo ,简单的增删该查,具体的springmvc还在探究。
如果只想让程序暂时先运行起来，可把spring-mybatis.xml中的内容<bean>注释掉，在运行。当然这是在基本配置都成功以后.
